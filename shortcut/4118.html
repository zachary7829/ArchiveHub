Reply · 6 months, 1 week ago
@robric18 Author
Yes it is very easy to select the parts of the shortcut you want to copy but it takes a bit of work before running the shortcut. Go to the shortcut you want to copy from. Put a comment before and after each section you want to copy. In each of those comments place three stars (like this ***). Anything contained within those comments will be copied. Be sure not to leave a hanging star comment or it will pick up everything after that comment.
Reply · 6 months, 1 week ago
@robric18 Author
Pro tip, if you want to paste the copied content in multiple locations, place the paste comment (a comment with three plus signs, like this +++) in each location you want to paste the copied items. It will paste the copied text anywhere in the shortcut where you place those comments.
