1.5 4 months, 4 weeks ago
iOS 13
Introducing UpdateConverter 1.5! This update is an exciting new update that integrates a special version of RouteUpdate in UpdateConverter. That means all you have to do is rename UpdateConverter to a different updater, and UpdateConverter will run instead without issues! (Note: Make sure the Updater that’s being replaced is compatible with UpdateConverter. Check the description for compatible updaters) UpdateConverter 1.5 also makes it so that it auto-adds your shortcut to UltraUpdate’s CFU Engine, and Swing Updater’s SFU! This means you can use Swing Updater’s Check for Updates feature, or UltraUpdate’s Check for Updates feature even if you don’t update with them! This update also fixes a bug with Change Updater (Thanks @ROP) Fun fact: For this update I had to rework the entire dictionary, which consisted of 77 magic variables :O This is a major update to UpdateConverter!
