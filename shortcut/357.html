Description

Photo to Text

Converts images to text

Uses OCR.space API to convert images to text. Get your free api key at OCR.space. Make sure the file is saved to your device before running the Shortcut.

Credits:

Thanks to Mike Beasley and Midnite for helping me understand how to get the JSON values.
