Description

What Can It Do?

  • Checks for an update once per day or check for them manually or every time the Shortcut is ran. Updates are always blazing fast using an Update System built by me
  • Checks if connected to WiFi to execute SSH commands
  • Locks the system
  • Unlocks the system
  • Change volume of the system
  • Pause/Play video (tested on YouTube and Netflix)
  • Restart the system.
  • Shutdown the system.
  • All user settings and passwords are saved in a file on the device for security.
  • All passwords are fully encrypted.
  • Add multiple profiles for multiple Macs.
