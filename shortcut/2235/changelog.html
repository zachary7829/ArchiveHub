2.11 4 months ago
iOS 13
Introducing Ultra Low Power Mode 2.11! This is a minor update to Ultra Low Power Mode. Here’s a changelog:
•The Shortcut is over all more optimized for better performance
•Removed Re:Updater Support.
•Replaced Swing Updater support with QuickUpdate, as QuickUpdate has support for multi-decimal versions, fully worded versions, a Skip This Update button, a vCard Menu, support for UltraUpdate, and more in just about 25 actions, which is over 400 less than Swing.
•QuickUpdate now required
•Some nasty bugs have been fixed
•Shortcut no longer exits, but stops on it’s own.

2.1 6 months ago
iOS 13
Introducing Ultra Low Power Mode 2.1! This is a major update to Ultra Low Power Mode, with lots of changes!

Changes:
•ULPM now changes to Dark Mode when enabling it. This allows you to save even more battery with it.
•ULPM now actually has a good GUI
•ULPM does not have auto-update for 13.4b1 devices, so it’s compatible with 13.4b1.
•UpdateConverter has been switched with qwertyuiop’s Re:Updater. This is so you can choose more updaters.
•Only compatible with iOS 13 now.
•CFU Support removed, as CFU is now discontinued and does not work for iOS 13.
•UltraUpdate CFU Support removed, as Swing will auto-support SFU.
•UpdateKit has been switched with D3W10’s Swing Updater. This is so Ultra Low Power Mode has a modern updater and is compatible with a modern CFU alternative that works for iOS 13.
•Custom Theme Support Added
•Now users can delete all ULPM2.1 data from Settings.
•Back To Normal renamed to “Turn Off”
•A function to change what appearance your device will have has been added to settings.

2.0 10 months, 2 weeks ago
iOS 12
It’s finally here - Ultra Low Power Mode 2.0. ULPM2 has a brand new UI, coming with a light & dark mode. It also fixes all of the bugs with Back To Normal. Another thing is that UpdateConverter support has been added, and Semi-UltraUpdate support has also been added. And, even with all of this new stuff, ULPM2 is only 137 actions - below 1.9’s 149, and 1.8’s 165! Technically this was 2.0 Beta 4, but, since there were no bugs, 2.0 Beta 4 has been publicly released as 2.0. This update was made by @zachary7829.

1.9 10 months, 2 weeks ago
iOS 12
Ultra Low Power Mode 1.9 fixes several bugs, including: Delete Reminders Not Working, New Reminders each time Shortcut is opened, wrong version in dictionary, Notification not showing, Turn Off Semi-Fixed (Brightness doesn’t work, Volume only works for some users, but everything else is fine). Reminders have been scrapped for Files. Now Boots slightly Faster. Actions have been reduced from 1.8’s 165 to 149. Better arrangement for the shortcut to feel faster. Now Supports iPod Touch. The only known issue is AutoUpdateData.json, however even that was made less irritating by not creating a new file each time. This update was made by @zachary7829.
