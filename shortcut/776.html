Description

Quickly add your work schedule to your calendar each week.

Features:

  • Simple creation of Calendar events for your work schedule.
  • Works with either "This Week" or "Next Week".
  • Quick input of Monday through Saturday (Can be adjusted to include Sunday if needed).
  • Import questions to select optional notes*, what calendar to use, and more.

Why this shortcut?

Personally, my schedule changes quite frequently as it depends on clients, so my shifts may or may not exist on certain days, and the times are almost never the exact same. Inputting these into my calendar manually then each time was a huge hassle, so I decided to help automate it with a shortcut, and figured other people may be interested in such a thing as well.

*Optional Notes, if enabled from import questions, will default to prompting an input box for what to put as the note. You can edit the shortcut as you wish to change this functionality to a preset note every time, or even something more complex, like selecting options from a list. You can easily find the section to edit as it's marked by a comment with a brief description on the note.

Automatic Updates are powered by UpdateKit & RoutineHub. You will need UpdateKit v2.0 or higher for them to work, although if you don't have UpdateKit at all, it just won't check for updates ever.

Latest Release Notes

1.2 - Dec. 7, 2018, 4:01 a.m.

- Fixed an issue where it wouldn’t tell you what was added to the calendar in some situations, instead asking if you skipped stuff. This also removes the notice that it’s added nothing, instead just showing a blank section for what was added.
- NOTE: Updating to this version will erase any changes and personal info from your current version of the shortcut. You’ll have to redo the changes and input your info again.

Yeah, of course it's possible - there's a few options depending if you want it to always be all-day or if you want it to only sometimes be all-day event. If you want it to always be, simply go in and change the Add Calendar Event action to toggle on the All Day option, then delete the questions about time. You'd probably need to remove the actual mentions of time from the "start and end" date spots in the Add Calendar Event also and then adjust the confirmation text to say all day if you want.
