Description

Welcome to RoutineFeed!

Well, what can I say about that?

It is a shortcut that simply mimics the visit to the "routinehub.co" website. The goal was not to create a version of a copyright infringement, but to create an - if possible faster or more convenient -alternative visit option for routinehub.co .

One of the main problems here was the speed. Because you can make a "get .. from .. HTML" version, but the resulting data consumption is relatively nogo.

So the idea was to create a "local storage" based variant. While experimenting with vCard markdown as a load and save option, I found out how quickly Siri Shortcuts reach their limits. Fantastically, working with .vcf files is like clockwork. 4000 files - no problem. The database is therefore based on Archiv.zip and must be downloaded at the beginning. (38.5 MB)

saved into Shortcuts/RHFeedDB/Merge/

Once downloaded, each time the shortcut is started, the last element is loaded from both the .zip and the website and their IDs are compared.

If there is a difference, ONLY the missing shortcuts are reloaded and added to the archive.

If there is no difference, the main menu loads and you can choose between the following functions:

  • Search (search the database = 3427 shortcuts (at the moment))
  • Feed (obviously)
  • Recently Updated Shortcuts
  • Trending shortcuts
  • RemindHub (just a "save-for-later list")

One more thing about the Archiv.zip file - the database

The profile picture of my account on routinehub.co is a QR code - which is captured and decrypted by the shortcut. The decoded content is the download link to the .zip file.

As far as I can, I always create an up-to-date QR code of the latest Archive and replace the profile picture on my account. This ensures that the latest version can always be loaded without having to update the shortcut every time.

Ultimately, it is just a fun shortcut and was a project I wanted to achieve and I am proud to have done it. Maybe it is used or even optimized by one or the other, the right to use is granted to everyone.

Thanks and good night,

GrotRec Shortcuts
