Description

There are more media download Shortcuts, this one combine a few of them. Supports YouTube (Video/Audio), SoundCloud (Audio), Facebook (Video/audio), Twitter (Video/Photo) and InstaSave (Instagram Video/Photo profile downloader).

This is inspired by SMD (Social Media Downloader, YouTube Converter, InstaSave and SoundCloud downloader.

Credits go to the creators of this shortcuts.
