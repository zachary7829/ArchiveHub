Description

Read. Annotate. In Shortcuts.

Overview

Annotate Is a Shortcut That Has Functions That You Do on a Book (Example Highlighting a Word, Taking Notes... digitally) and Many Smart Functions That You Couldn't Do With a Book and Pen (Example Learn Pronounciation, Search on Different Platforms and Dictionaries, Listen the Book...)

Setup

The setup process may take 1 minute.
1. We’ll ask the Book Name
2. We’ll ask the Author of that book
3. We’ll ask the Book Pages
4. We’ll ask Three More questions to Personalise Annotate
Setup Complete!

Notable Features

  • Highlighter with Highlight History and Review Highlights in One Single Place
  • Search with Many Different Search Engines, Platforms and Different Dictionaries
  • A Very Detailed Stats of Your Reading Progress
  • Learn to Speak Words — When you Saw a New Word, and you Want to Know How to Read, we will Teach You How to Read it, and You Can Practise and Interact with Our Virtual Teacher
    ... and Much More!

Important

Permissions

We Need Access to iCloud Drive to Save Reading Progresses, Highlighted Words, Dictionary History, Annotate Settings... We won’t be able to read data other than Annotate’s data.
We also Need Access to Website (RoutineHub) to Check for Updates available for Annotate. Other than RoutineHub, we Will Not access to other Websites and Send Data or Download Data to/from any Websites.
If you still have any permissions questions, go to feedback below this Description. You MUST allow Annotate to Access to iCloud Drive and RoutineHub, or else Annotate won’t Work Properly.

Data

We Are Not Liable to any Data Lost and any Other Data based Problems.

Compatibility

Requires iOS 13.0 or later. Compatible with iPhone, iPad, and iPod Touch.
