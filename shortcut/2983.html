The First EVER Updater for iOS 13!

DOCUMENTATION AVALIABLE (Includes Instructions for iOS 13)

UpdateBot

The best Updater that works seamlessly for anything! UpdateBot will seamlessly work with your shortcut unlike many other "drop-in" updaters. The unique part about UpdateBot is that UpdateBot will work for both RoutineHub and Shareshortcuts. No longer do you have to use the default Shareshortcut updater. Different versions no more! You can now have different versions of your shortcut in Shareshortcuts and RoutineHub and STILL have it be updated!
