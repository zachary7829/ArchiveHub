Welcome to
PDF to Grayscale

This shortcut converts a colored PDF file into a grayscale PDF. For this purpose, the individual pages are extracted and converted into image files. In a further step, each page is converted to grayscale using JavaScript. Afterwards, all images/pages are converted back to PDF.

 
 
 
 
 
 
 
 

 
 
 
 
 
 
 
 

How To

The shortcut can be used via the Share Sheet or directly in the Shortcuts App.

1. Select a PDF file from iCloud Drive.

2. Now you can choose whether the whole PDF or only individual pages should be converted to grayscale.

3. The shortcut now generates a grayscale PDF. Please be patient. Depending on the number of pages it can take quite a long time until the conversion is completed. So please consider carefully if you really need all pages.

4. The PDF can be saved in iCloud Drive or in your Dropbox in the last step. Alternatively, you can share it and pass it on to other apps. If you choose "Show PDF", you will also have the option to save or share it afterwards.

 
 
 
 
 
 
 
 

Credits

The JavaScript code used in "PDF to Grayscale" is based on the JS code of "Invert Image" shortcut by @keveridge.

 
 
 
 
 
 
 
 

No third party Updater Shortcut required.

PDF to Grayscale has an integrated update mechanism called Embed-a-Update. Thanks to the embedded updater, PDF to Grayscale does not require any external updater shortcut. The shortcut checks for updates itself, so you'll always be up to date and never miss an update.
