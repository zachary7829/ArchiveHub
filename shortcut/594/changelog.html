3.0 2 months, 4 weeks ago
iOS 13
Snapcut for iOS 13 is finally here!

Apple made some changes in Shortcuts app in iOS 13 that affected some details in Snapcut 3, but it has the same features of Snapcut 2 and more. With Snapcut 3 you can now set the size and aspect ratio of background images.

Release notes:
- Built using iOS 13 Shortcuts app;
- Added an option to set the size and aspect ratio of background images. Choose between 1:1, 4:5, 3:4 and 9:16 for aspect ratio, and Low, Medium and High for resolution;
- Solved a problem that leads Snapcut to run indefinitely when add a text containing specific emojis, including new emojis added in iOS 13.

2.0.1 1 year, 3 months ago
iOS 12
- Snapcut 2 uses the vibrant.js library to create gradient background from images. This update includes the vibrant.js license.
- Snapcut 2 now requires UpdateKit 3.0 or later.

Snapcut 2 release notes:
- Add text to your photos: You can add multiple texts with 11 different color themes to choose from. More themes will be added in future updates;
- Photo position options: You can select common positions to place the photo. The options are: show entire photo, partially crop and crop to fullscreen. You can also place the photo manually if you prefer.
- New background styles: Gradient background, Black background or No background. Gradient background automatically select colors from photo to create a nice looking background. Black background can be used when you want to post only text or sticker. And you can choose No background when you want add text or sticker to your photo without crop or resize it.
- New sticker options: You can choose sticker from Bitmoji Keyboard or any image from your clipboard. When you search a sticker in Bitmoji Keyboard, Snapcut ask you if you selected a bitmoji because you could not found any bitmoji you like.
- Small changes and improvements.
