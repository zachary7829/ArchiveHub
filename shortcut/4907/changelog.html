1.34.0 2 months ago
iOS 13
-Removed mandatory passwords. Now you, and only you, decide what your passwords are. You can now change old mandatory passwords.
+Added Facade passwords. Like a ‘safeword’, entering this password lets a compatible shortcut know that you are being watched or controlled, and allows them to create a fake, safe version of the shortcut for prying eyes. When an existing shortcut updates to add compatibility here, you will have to delete the save file to set a facade password, however, if you don’t want a facade password, the old save works fine.
