2.0 1 week, 4 days ago
iOS 13
Pictionary 2.0 introduces a really awesome new feature - words are now discarded after they’ve been used in the current game! That means no more repeated words in the same game! I struggled for a long time with how to achieve this, so I’m quite proud of myself for pulling it off.

Special thanks to @robric18 on Reddit for their tips on how to make this feature a reality. They’re the creator of 50 States Plus, another shortcut game available on RoutineHub.

Also in version 2.0 comes 50 additional words, for a total of 389! Finally, I’ve made a few smaller quality-of-life improvements, like a “Return to Shortcuts” button at the bottom of the Rules & How to Play screen, a new color for the shortcut, etc.

Note for Beta testers: Pictionary 2.0 may or may not work as expected on iOS/iPadOS 14 Betas. As of Beta 3, there’s a bug that causes this shortcut to present an odd notification and then freeze if a timer is running when the shortcut attempts to override it. Also, expect to see a lot of “Shortcuts pasted from” notifications. This is an intended behavior, and is part of how the shortcut discards repeated words, and part of how the shortcut knows if you’d like to open the drawing board. If Pictionary 2.0 isn’t working properly on your beta testing device, you can let me know by using the send feedback button, and I’ll do my best to fix it, if possible.

1.5 2 months, 3 weeks ago
iOS 13
Now you can draw directly inside the shortcut! Version 1.5 adds a drawing board option (you can still play the same as before if you prefer that), with full Dark Mode support! It scales for both iPhone (portrait) and iPad (landscape), although on iPhone its pretty impractical, to be honest. But it’s there if you want it. The drawing board does take a few seconds to appear, so I made it reset the timer to 60 seconds as it displays the drawing board, leveling the playing field as much as I reasonably can. (This actually gives a slight advantage to the slowest devices, as they have more time to think!) Credit to u/bamfhacker and u/vbrider on Reddit for suggesting this addition.

Oh, and I added a word. And now it stops when it tells you that you’ve won, instead of continuing for no reason.
