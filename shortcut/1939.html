Description

 

Introduction

Always wanted to have a shortcut that does a lot of stuff and it's easy to use and never found one? Utilities Menu (or UM) could be the answer for that problem. With it you have access to more then 100 utilities that you can try and explore.

 
 
 
 
 
 

Text and Numbers

In this category you can find all the utilities related to text and numbers like Translating stuff, apply effects to text, discount calculator, etc.

 
 
 
 
 
 

Media

In this category you can find multiple utilities related to media like creating Instagram Panoramas, read QR codes from photos, check the daily word/image and even upload pictures to imgur!

 
 
 
 
 
 

Web

Web category allows you download videos from YouTube, expand urls, see old versions of websites and you can even access to Safari Tools where you can find things like Auto Scroll Mode and Webpage Font Changer.

 
 
 
 
 
 

Entertainment

If you want to have fun as you don’t have nothing to do, entertainment category besides having Apple TV control, IMDb Rating and Sound Gallery, it has too some games that you can play even if you are offline!

 
 
 
 
 
 

Shortcuts

This category helps you manage and develop your shortcuts as it features tools like Get Shortcut Icon, Shortcut count, See Shortcut Plist and Shortcuts Backup.

 
 
 
 
 
 

Other Stuff

For utilities that doesn’t fit in any category above they will appear here. Here you can find Countdown, File Viewer, Internet Password Share, Utilities Menu Vault, Water Eject and even more!

 
 
 
 
 
 

Settings

All the Utilities Menu Settings can be found here, as well as Send Feedback and Check For Updates button.

 
 
 
 
 
 

Dashboard

Here you will find controls for basic applications like Messages, Phone, Music, Maps, Weather, Calendar, Reminders, an Applications category and Switch Settings. On Applications you can open apps and download AppStore artworks and in Switch Settings you can change a lot of iOS Settings.

 
 
 
 
 
 

COVID-19

As we are in a pandemic season, Utilities Menu temporary added a COVID-19 menu where you can find Apple Screening Tool and all the known information about the virus.

 
 
 
 
 
 

Update Checking

To update Utilities Menu, you will need to install Swing Updater to always stay with the latest bug fixes and features.
