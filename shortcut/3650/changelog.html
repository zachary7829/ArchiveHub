2.00 4 months ago
iOS 13
Version 2.00
Version 2.00 is here. Finally bringing this shortcut in line with the Sonarr shortcut in terms of features and menu options. I've lost track on exactly everything that changed. But here are the high lights.
-Added queue menu. View download queue or view download history for the past 10 items.
-Renamed edit menu to 'advanced'.
-Added caching of movie menu items. This greatly improved the speed for searching for movies. And accessing any menu that required a compiled list of shows. Which means fewer repeated steps in each menu. Movie cache will update periodically and when adding a new movie.
-Added sorting of selecting releases based on quality weight, this should place the more relevant releases on top of the search results.
-Fixed incorrect sourcing of key when searching for movies. Therefore fixing returning a ton of unwanted profiles.
-Added rich notifications in all notifications that support it.

Version 1.50
- Added prompt to show movie description when adding a new movie, this will allow to confirm you selected the correct movie before adding it to Radarr.

1.10 10 months ago
iOS 13
Version 1.00 of Radarr

- Add movies to your library
- Automatic search for movie files
- Selective search for movie files
- Pretty menu on searching, listing out your current library of movies, and detailed information of what file you are selecting for download
- An easy to follow setup will run on first run of shortcut.

1) If you want to add 4K movies
2) API Key. Found in settings-general-API
3) Sonarr url This should be set to ‘http(s)://IPorHOSTNAME:PORT/api’ if no base URL. http(s)://IPorHOSTNAME:PORT/baseURL/api’. If you use a reverse proxy ‘http(s)://IPorHOSTNAME/baseURL/api’
4) root path where you store you HD quality media content. i.e /media/Synology/Content/Movies
5) root path for where you want to store 4K quality media, can be different or same as above.

I will be adding in more feature in the next week or so.
