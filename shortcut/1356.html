Description

 

 
 

With Request Desktop Site you‘re able to force the Desktop View of every website—even if this does not work with the dedicated Safari button.

 
 
 
 

Simply visit a website with Safari, open Share Sheet, scroll down, tap Request Desktop Site and allow access (don’t worry, no information will be passed on to third parties). That’s it!

 
 
 
 

 
 
 
 
 
 
 
 

No third party Updater Shortcut required.

Request Desktop Site has an integrated update mechanism called Embed-a-Update. Thanks to the embedded updater, Request Desktop Site does not require any external updater shortcut. The shortcut checks for updates itself, so you'll always be up to date and never miss an update.
