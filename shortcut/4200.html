Description

Welcome to RoutineHub Stats Advanced

This Shortcut was created to report the increments in downloads and hearts an author receives since the last time checked.

FEATURES

RoutineHub Stats Advanced features

  • Multiple Users
  • Total Heart count
  • Shows Last Checked
  • Line Charts through Charty (Coming soon)
  • Daily Average (Coming soon)
  • Integrated Self-update checking
  • And much more...

Set up

  1. Press 'Settings'
  2. Press 'Add'
  3. Search for your favorite authors (including yourself)
  4. Rerun the Shortcut and press 'Start'
  5. Repeat as many times as necessary

Startup Page

After you have finished setting up your user list, all you must simply do is press ‘Start’. You will be presented with a beautiful interface of your user list and will be asked to select a user from it.

Result

After you select a user from your list, the shortcut will open up a list view report with Light/Dark mode support. It will present the user's shortcuts increase in downloads and hearts since last checked.
