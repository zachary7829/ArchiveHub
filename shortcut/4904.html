Description

Shazam → Spotify

This shortcut is an easy way to let Shazam identify the currently playing song and add it to a predefined Spotify playlist.

Until today Spotify doesn't really offer any useful shortcut support. Adding songs to a personal Spotify playlist is quite cumbersome when using their official web API. That's why I chose to use IFTTT for that purpose which makes the whole process a lot easier.

You need to create a little IFTTT webhook in order to make this shortcut work:

  • Login to your IFTTT account
  • Click on your profile picture (upper right) and select "Create"
  • If → click the "+" icon → choose "Webhooks" → Receive a web request → name it "add_track_to_spotify" → Then → Spotify → Add track to a playlist:
    • Playlist name → click "Add Ingredient" → choose Value1
    Search query → click "Add Ingredient" → choose Value2
    Artist name → click "Add Ingredient" → choose Value3

    Preview: It should look like this

To obtain your IFTTT webhook key:

on IFTTT click on your profile picture (upper right) → My Services → choose Webhooks → Settings (upper right) → The key is the last part of the URL, in my case 2LbE..., see:

→ This key is needed for this shortcut and the import wizard asks for this key upon installation.
