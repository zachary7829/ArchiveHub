Description

Re:Updater is a shortcut that allows you to use any Updater that the user has chosen. Re:Updater is not an updater, it is the middle between your shortcut and the updater. The advantage of this shortcut is for the user, it avoids having several updaters doing the same thing.

Updater supported:

Updater in-Shortcut:

  • Re:Updater

Updater 3rd party Shortcut:

  • UpdateKit
  • JustUpdate (compatible with JustUpdate Lite)
  • Swing Updater
  • iGotUpdate
  • LightningUpdate
  • Updates Check Auto
  • QuickUpdate
  • PeeWee Update
  • EasyUpdater SF
  • iUpdate
  • Speedy Update
  • UltraUpdate
  • UpdateR
  • UpdateCorner
  • Swing Updater [S]
  • Control Centre Update - Download fixed version
  • Demon Updater - Download fixed version
  • Update Checker - Require minimum 5.9
  • MyUpdate
  • UpdateFaucet
  • iUpdate LSE
  • R⤓Updater
  • Shortcut Updater
  • FastUpdate
  • UpdateLab
  • QuickUpdateEX

Important!!! : For Demon Updater and Control Centre Update, you have to download the version in the description, because the original version is broken.

* Yes there are probably some updaters that are not yet supported by the shortcut, here is the status of the remaining updaters:

  • UpdateHub: Impossible, because it analyzes the shortcut code (that's what I think) and since there is no "static" information and Re:Updater use variable to redirect information to the good updater, it's just impossible.
