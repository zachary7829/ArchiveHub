2.3 7 months, 2 weeks ago
iOS 13
Sorry for the late update everyone, but I wanted to make sure everything was set up smoothly and the changes worked fine. Anyway, check out what’s new in the biggest Minecraft Studio update this year!:

• Welcome message when downloading the shortcut.
• Mojang API section in Other Resources, currently you can check the API status. More options for the API coming in 2020.
• Nether Portal Calculator, which debuted early yesterday, is now built-in with all of its features!
• Slime Chunk Finder option in Other Resources, which leads to an online slime chunk mapper for bedrock edition. Useful for slime farms.
• More Documentation section design and glyphs updated, now reflects design style of main menu. Added Search Minecraft Wiki option.
• New Settings option in main menu, where Check for Updates is now moved. More options coming in 2020.
• Minor bug fixes and improvements.

Well, that’s all folks! This shortcut has sure come a long way since the first release in March! Thanks for all the support, it really helps me stay motivated to push updates! Here’s to the good parts of 2019, and to an even better 2020! ☃︎
