2.0 6 months, 1 week ago
iOS 13
Bug fixes:

- During generation of more than one class loadout, as soon as a Shotgun was randomly chosen for the Primary Weapon, Primary Weapon Scope was never shown for any other Primary Weapon that isn’t a Shotgun. This has been fixed!

- Perks and Scorestreaks have been squashed of a appending variable bug.


Quick Easter Egg:

- If you don’t input a number for how many classes to generate, the shortcut will automatically generate 9 random classes anyways. This isn’t necessarily a bad thing and I don’t think it causes any problems. It’s a nice thing for debugging any issues found while working on the shortcut.
