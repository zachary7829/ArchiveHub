9.0 4 months, 2 weeks ago
iOS 13
• Swing Updater has brand new theme, Remix and, it’s icons shine Swing Updater simplicity and power.
• Removed Swing and Modern Themes.
• Swing Updater, for the first time ever, supports ShareShortcuts, giving you more choice on where to publish your shortcut.
• And for those who think in posting it’s shortcuts by themselves, Swing Updater supports Custom Servers, giving you all the versatility.
• Replaced the Swing Integration Example button with Download Center, there you can find the links for all shortcuts related to Swing Updater.
• Settings now features a Send Feedback to report bugs, send suggestions or just leave a kind word.
• Settings now contains Icons, looking a lot more friendly.
• The Remove all Skipped Versions button in Settings has been replaced by Remove Skipped Version, didn’t get the difference? Now you can remove individual Skipped versions instead of removing them all!

iOS 13
