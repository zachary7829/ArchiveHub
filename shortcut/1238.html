Description

With Turn on my TV you're able to boot/turn on/wake your tv and TV (HDMI-CEC required) at the same time. This works for tvOS 13 and above.

 
 
 
 

 
 
 
 

 
 
 
 
 
 
 

Import Questions

After importing the shortcut, you will have to go through a configuration process:

  1. This shortcut uses an embedded updater to check for updates. So when you run it for the first time in the Shortcuts app, please allow access to RoutineHub.co. You only have to do that once.
  2. To run this shortcut, you need a TV that supports HDMI-CEC, an Apple TV and you have to be on the same Wi-Fi network as your Apple TV. Furthermore you need to grant access to Apple TV Remote. In the second step of those five Import Questions please select your Apple TV.
  3. Do you want the Apple Remote Control app to open after your Apple TV and TV are turned on/woken up? Type "Yes" or "No".
  4. No matter which answer you chose in the previous step. Please select your Apple TV again. If you chose e.g. "No" in step 3, but want to change this setting to "Yes" later, that's no problem. All you have to do is return to the Import Questions: Open the shortcut in edit mode and tap the button with the three dots in the upper right corner. Then tap on "Customize Shortcut".
  5. One last thing: With iOS 13 it is no longer possible to create your own Siri phrases like in iOS 12. Now with iOS 13 the name of the shortcut is also the Siri phrase. Since the name of the shortcut is "Turn on my TV", you have to activate the shortcut as follows: "Hey Siri, turn on my TV". If you want to use another Siri phrase, you have to rename the shortcut.

 
 
 
 
 
 
 
 

 
 
 
 
 
 
 
 

No third party Updater Shortcut required.

Turn on my TV has an integrated update mechanism called Embed-a-Update. Thanks to the embedded updater, Turn on my TV does not require any external updater shortcut. The shortcut checks for updates itself, so you'll always be up to date and never miss an update.
