4.2.7 1 month, 3 weeks ago
iOS 13
Change Log (v4.2.7):
•Replaced Embed-a-update with a custom Embedded Updater
•Updates can now be searched much faster
•Added vCard Icons to Update Prompt
•Now Update Release Dates are shown
•Fixed issue when No Available Links are found, the Shortcut was abruptly stopped
•Fixed My Other Shortcuts not being Downloaded Properly
•Bug fixes & Minor Changes

Change Log (v4.2):
•Added ‘My Other Shortcuts’ to Options
•Added Fancy vCard icons to Social Media and More Links menus!
•You can now rearrange the order of ‘More Links’ using the Platforms dictionary
•Added Back button to More Links.
•Now the Correct User Country Is always used when finding Song Links
•Various Bug fixes and Optimisations

New Features (v4.0):
•Biggest Update Yet
-Completely Overhauled Design and Revamped Menu Navigation
•Introduced About Me Option
-Shows Details about the Shortcut
•Links now Open Directly
-Improved links on YouTube/Spotify etc. to open the song directly rather than only searching for the song
•New View Artist feature
-Discover more from the Song’s Artist and quickly find their Wikipedia, Social Media, View their Other songs or Share.
•New More Links
-Choose from a list of every platform a Link to the song was found on without visiting an external site
•New Share Menu and Updated Options
-Includes Send via Messages/WhatsApp, Add to Notes and More

Change Log (v4.1.7):
•Fixed issues with Social Media feature
•Lowered default Artwork resolution to 2000 to reduce storage size
•Doubled results of Other Songs shown by Artist (Now 40)
•Bug fixes and optimisations such as reduced actions and increased efficiency

Change Log (v4.1):
•Removed Sound from Notification as some users reported it can be annoying after time
•Updated Apple Music URLs and Video URLs to directly Open In-App
•Improved Speed when sharing URLs from Share Sheet
•Added much Higher Resolution Artwork which can be customised and comes with a suitable name when viewed
•Reintroduced Add to Playlist for Apple Music Option
•Added option to Save Artwork to Camera roll after Looking at Artwork
•Improved speed of About Song Menu
•Massively improved speed of using More Links through a New Dictionary
•Improved If Statements at the start to more accurately determine if a valid Music URL is input
•Now you are able to Share ANY Music URL to find Links (Previously only supported Apple Music)
*This Last change introduces several new possibilities e.g. you can now Share a Spotify URL which you can open in Apple Music if you select Shazam++ from the Share Sheet etc. I recommend you mess about with feature to help your own personal use cases.
•Apple Music URLs from Song.Link are tweaked to Open faster
•Social Media improved to show results within Shortcut, now a lot more useful.
•Apple Music Option can now search iTunes Store if no link was found
•Minor Bug fixes and changes
