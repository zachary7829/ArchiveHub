Description

Remindly

Simple, fast reminder management

Remindly has a few opinionated options geared toward speeding up my Reminders app workflow, and I hope it helps you as well.

Features

With Remindly, you can:

  • Create new reminders
    • create any number of reminders in a row without interruption
  • View upcoming reminders
    • from a specific list
    • for any number of days in the future
    • for any specified location
    • all upcoming reminders regardless of due time or location trigger
  • Manage overdue reminders
    • View a list of overdue reminders
    • Browse and Delete reminders
    • Delete all overdue reminders
  • Remove all reminders that match a filter:
    • By Name / Title
    • From A Specific List in Reminders.app
    • Before Due Date
    • Before Completed Date
    • Without an alarm (no time or location based trigger)
    • Clear all overdue reminders
    • Clear all completed reminders

Remindly also has a Settings menu where you can

  • View the current settings and shortcut info
  • Manage UpdateKit
    • Manually check for update
    • Toggle Auto Update (default: off)
      • update whenever Remindly runs
      • auto update will cause Remindly to run a little slower
  • Manually reinstall from iCloud
    • with the option of deleting settings before installation for a fresh install
  • Delete all settings
    • useful for troubleshooting / starting fresh


Coming Soon

  • Auto-clear completed reminders once per day (via Settings toggle)
  • "Favorite Locations" feature that will allow saving of a few default addresses selectable when creating a new reminder
  • Add Alarms and Timers to the "Create" menu


Bugs / Issues

There may be menu item changes and shortcut structure improvements while I work on adding features. The core functionality of the shortcut will remain the same. This should level out as I get closer to version 1, and apologies in advance for any issues or confusion.
