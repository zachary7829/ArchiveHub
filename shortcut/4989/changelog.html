3.5 4 days, 3 hours ago
iOS 13
What's new in version 3.5

- the trigger for this shortcut has been changed just a bit. It has been changed to "Manage Home Bridge!" instead of "Manage Home bridge!". This should resolve some issues for HomePod users that resulted in "I found a few locations matching Home Bridge" as a result, you will need to delete the old shortcut.

- Uptime is now a feature! Ask it how "Long" the server has been online! She will report the time in either days, minutes, or hours (not a mix). This feature is beta so let me know!

- a few command changes for internet tools. If you wish to do an internet connection check, just say "connected (or connection. Anything that says "CONNECT" in the name) and Internet. For an ISP speed test, just say "speed" and "internet" in the same command.

iOS 13
Version 2.7 Release Notes

What’s new:
-Pi users can now get a CPU temperature! (So the legend has foretold!) MacOS users should still be able to do the same, but they will need the home brew dependency (sox-cpu-temp)
-If you have configured this shortcut before this version, you will now be asked to add a platform for your instance! This will replace the old configuration :)

Fixes:
- I have changed some of the tools in the help me command.
- when restoring the shortcut, the trigger word is now “restore” instead of “shortcut”
- the shortcut now will tell you if it was unable to restart the Home-bridge service. If this works out, I’ll apply this to rebooting :)
- fixed some of the feedback to be a little more varied and natural.

2.0 2 months ago
iOS 13
Version 2.0 Release Notes
- this is the current release! We have added some changes that I hope you will understand.

-you now have the option to use SSH keys. You will add them in configuration :)

- internet speed test! You can now test your download (should you wish) this uses a brew tap Speediest-cli. You will need it installed.

You can now get your cpu temp! This uses a brew tap called OSs-cpu-temp

I have decided to start taking advantage of the config-ui-x api! As such, figuring out the running status of Home-bridge is more accurate :) this will require you to set up an API key. The shortcut will tell you how on import!

-remote tools have been removed. They will be added back in at a later date, but given the use case of this shortcut, I found that it works better if everything is ran locally anyways.
