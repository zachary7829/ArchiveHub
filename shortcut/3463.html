Description

Welcome to UpdateConverter!

UpdateConverter is a shortcut that allows you to use any Updater you want! You add a dictionary to it, and UpdateConverter converts that dictionary and runs the Updater that the user chose to run. (Note: Please keep in mind UpdateConverter only adds 0.05 - 0.2 Seconds to the updating process.)

Works on iOS 12 & 13!

UpdateConverter has been tested, and confirmed to work on both iOS 12 & iOS 13! (Thanks to @JuhJuhButts for testing iOS 13 support)

UpdateConverter 1.0 - 1.1.1 (iOS 12 & 13)

UpdateConverter 1.2 - 1.6.1 (iOS 13)

How to Add Support:

UpdateConverter’s dictionary is the Same as UpdateKit’s dictionary, so you can easily switch your shortcut from UpdateKit to UpdateConverter.

 

Shortcut Name - Name of your Shortcut

Current Version - The version of your shortcut

RoutineHub ID - The ID of your shortcut

Switch to UpdateConverter

UpdateConverter has a special version of RouteUpdate integrated inside. This means all you need to do is clone UpdateConverter, rename it to the updater of your choice, and it will run instead! (This feature was inspired from SwingPlug, by @Burrito) All 33 updaters UpdateConverter supports are compatible with this feature!

Add Custom Updater

With UpdateConverter, you can add your own custom updater, even if it’s not yet supported by UpdateConverter!

Nice and Speedy.

UpdateConverter is less than 100 actions. That’s incredibly fast, and means lower latency!
