What's new: added the ability for you to share your Shortcuts. Just take a shortcut from the list of Shortcuts that you want to share, then it will copy the link to your shortcut to the clipboard. after that you will hear a cool sound After you share the link.
This shortcut now gets your device details when the prompt says hello device name, welcome to all in one shortcut. Instead of when you run the shortcut instead of saying hello welcome to All in one shortcut, it will say your devices name.

1.3 9 months ago
iOS 13
There are a few changes to keep in mind for this update. First off, you can listen to 100 to 7 kiss FM Los Angeles using iHeartRadio.
The record audio has been changed. Instead of recording using the Shortcuts application, it uses Voice Memos.
The random shortcut has been updated as well. In order for the shortcut to work, you must get the flash shortcut, which message wrong with the light settings and other stuff. You can get it here.
https://www.icloud.com/shortcuts/92f2bdfafd6e4642afd56006bd94bfce
When you get the flash shortcut, the random shortcut will run the flash shortcut. Due to the random joke API not working, the random joke option is removed. otherwise, everything else is the same. Leave feedback if you have any. Don't forget to request some features and stuff.
