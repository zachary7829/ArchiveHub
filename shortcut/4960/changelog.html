iOS 13
Introducing 2.3! This update mimics PassKit’s file saving to make it even harder for a potential DRM to detect it against regular PassKit! In other words: a shortcut could have checked if a file was present after PassKit was ran to see if it was the real PassKit, but that has since been fixed in this update.

2.2.1 3 months, 1 week ago
iOS 13
A shortcut that supports PassKit could potentially check PassKit to see if it was below a certain file size, and if so, it wouldn’t work. A shortcut could also check how many actions there are to see if PassKit was the “bypass”. However, both of those things for a potential DRM to check for have been fixed, with a bunch of nothing actions and 0’s.
