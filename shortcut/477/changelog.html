3.0 1 year, 9 months ago
iOS 12
- Caching enhancements

Caching now caches more than the current rotation period. When running the shortcut, it checks for a cache. If there’s a cache, it checks if any of the 12 cached periods is the current period. If so, it uses that data, and if it’s not cached it fetches new data and caches the new data.
