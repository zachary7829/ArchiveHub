Description

 

Yep, you read it right. You can search for ANY RoutineHub Shortcut or Author within this shortcut.

No

  • Leaving to the RoutineHub Website
  • Distracting Interface
  • Ads
  • Shortcuts only Search

Advanced Use

Inputs & Outputs

  • Search for Authors
    Input = {"Author":"Name"} Example Output
    {"Name:"Shortcut Name","URL":"RoutineHub Shortcut URL"}

  • Search for Shortcuts
    Input = {"Shortcut":"Name"} Output = {"Name:"Shortcut Name","URL":"RoutineHub Shortcut URL"}

  • Back Option
    Add in {"Back":"Input Shortcut Name"} to the input dictionary so that it returns the necessary information (the json of the Shortcut’s name and URL) to the Shortcut that it is running from.
